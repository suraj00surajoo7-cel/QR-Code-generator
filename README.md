# QR-Code-generator

A modern and responsive QR Code Generator web application built using HTML, CSS, and JavaScript. This project allows users to instantly generate QR codes for URLs, plain text, and contact information (vCard format). It features real-time QR generation, download support, clipboard copy functionality, and a clean user-friendly interface.

## Features
Generate QR codes for:
Website URLs
Plain text
Contact details (vCard)
Instant real-time QR generation
Download QR codes as PNG images
Copy QR data to clipboard
Responsive and modern UI design
Lightweight and fast performance
## Technologies Used
HTML5
CSS3
JavaScript
QRious Library
